# Changelog for WooCommercePS

The format is based on and uses the types of changes according to [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- Added function Get-WooCommerceCategory - Retrieve product categories
- Added function Get-WooCommerceCredential - Check for WooCommerce credentials
- Added function Get-WooCommerceCustomer - Retrieve WooCommerce customers
- Added function Get-WooCommerceData - Retrieve WooCommerce data/information
- Added function Get-WooCommerceSetting - Retrieve settings groups
- Added function Get-WooCommerceSettingOption - Retrieve a specific setting option
- Added function Get-WooCommerceOrder - Retrieve orders with filtering and sorting
- Added function Get-WooCommerceOrderNote - Retrieve order notes
- Added function Get-WooCommercePaymentGateway - Retrieve payment gateways
- Added function Get-WooCommerceProduct - Retrieve products with various filter options
- Added function Get-WooCommerceProductCustomFields - Retrieve product custom fields
- Added function Get-WooCommerceProductReview - Retrieve product reviews
- Added function Get-WooCommerceProductTag - Retrieve product tags
- Added function Get-WooCommerceProductVariation - Retrieve product variations
- Added function Get-WooCommerceRefund - Retrieve refunds
- Added function Get-WooCommerceReport - Retrieve a specific report
- Added function Get-WooCommerceReportList - Retrieve available reports
- Added function Get-WooCommerceShippingMethod - Retrieve shipping methods
- Added function Get-WooCommerceShippingZone - Retrieve shipping zones
- Added function Get-WooCommerceShippingZoneLocation - Retrieve shipping zone locations
- Added function Get-WooCommerceShippingZoneMethod - Retrieve shipping zone methods
- Added function Get-WooCommerceWebhook - Retrieve webhooks
- Added function Invoke-WooCommerceAPICall - Internal function for API calls
- Added function New-WooCommerceCustomer - Create a new customer
- Added function New-WooCommerceProduct - Create a new product
- Added function Remove-WooCommerceCustomer - Delete a customer
- Added function Remove-WooCommerceOrderNote - Delete an order note
- Added function Remove-WooCommerceProduct - Delete a product
- Added function Remove-WooCommerceProductReview - Delete a product review
- Added function Set-WooCommerceCredential - Set WooCommerce API credentials
- Added function Set-WooCommerceProduct - Update an existing product
- Added function Set-WooCommerceProductReview - Update a product review
- Added function Update-WooCommerceCustomer - Update an existing customer
